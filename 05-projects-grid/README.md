### 📦 Libraries I maintain

Small, focused primitives for LLM/agent systems.

| Package | What it does |
|---|---|
| [`palimpsest`](https://github.com/asmitdash/palimpsest) | Contradiction-aware agent memory with provenance-preserving forgetting |
| [`chronograph`](https://github.com/asmitdash/chronograph) | Temporally-aware RAG — every edge has a validity window + decay curve |
| [`replay-rag`](https://github.com/asmitdash/replay-rag) | Reasoning-trace memory for o1/R1-style agents — stop re-deriving |
| [`corroborate`](https://github.com/asmitdash/corroborate) | Deterministic answer-grounding, no LLM-as-judge |
| [`chaffer`](https://github.com/asmitdash/chaffer) | Lint for RAG corpora — duplicate chunks, dim mismatch, eval leakage |
| [`staleness`](https://github.com/asmitdash/staleness) | Freshness + drift linter for RAG knowledge bases |
| [`relapse`](https://github.com/asmitdash/relapse) | Self-correcting LLM caller with hallucination-aware retry |
| [`dash-mlguard`](https://github.com/asmitdash/dash-mlguard) | Lint for ML training pipelines — leakage, drift, schema mismatch |
